# 🚀 START HERE - ControlCajas

¡Bienvenido! Esta es tu aplicación web **ControlCajas** completamente funcional.

## ⏱️ Quickstart (5 minutos)

### 1. Instala

```bash
npm install
```

### 2. Población de ejemplos (recomendado)

```bash
npm run seed
```

### 3. Inicia

```bash
npm run dev
```

### 4. Abre

```
http://localhost:3000
```

### 5. ¡Usa!

- Clica "Regístrate"
- Crea tu cuenta
- ¡Comienza!
